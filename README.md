# Internship Task Submission - March 2026

This repository contains the tasks allocated for the March 2026 evaluation.

## Task 1: Personal Portfolio
A fully responsive personal portfolio website built to showcase projects, skills, and academic background.
* **Technologies:** HTML5, CSS3, Vanilla JavaScript
* **Features:**
  * Custom typing animation effect
  * Responsive navigation and CSS Grid/Flexbox layouts
  * Direct WhatsApp integration for the contact form

## Task 2: Task Tracker (To-Do List)
A browser-based task management application.
* **Technologies:** HTML5, CSS3, Vanilla JavaScript, LocalStorage API
* **Features:**
  * Add, edit, and delete tasks
  * Mark tasks as complete
  * Data persistence using browser LocalStorage

## How to Run
1. Clone this repository.
2. Open `Portfolio/index.html` or `Todo List/todo.html` in any modern web browser. No local server is required.
